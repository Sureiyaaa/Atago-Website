I. Introduction

About:

Atago is created for collecting waifus and for welcoming users to your own server! Not only that, Atago can announcing server boosts with customizable boost message!

Note: Atago Source Code is Private

Requirements:

Requirements for Atago to spawn waifus is to have 20+ members in the server. This prevents solo farming waifus in private servers. This only affects the waifu system, not the other commands.

Permissions:

View Audit Log

+ Optional! for Auto Role Feature.

Manage Emojis
+ Important! for Inventory Reaction and Leaderboard (Next Page)

Manage Messages
+ Important! for Fusing System and for removing messages of Atago.

Embed Links
+ Important! for Invite for Support Server and Inviting Atago to other Servers.

Attach Files
+ Important! for Images on Profile and Cards.

Connect
+ Important! for Music System.

Speak
+ Important! for Music System.

II. Functions

Waifu Collection:

Collecting waifus will only work if the members in the guild are sending messages. Every 15-25 messages there will be 1 waifu spawning in the waifu channel you have set.


Music:

Enjoy Collecting Waifus while listening to Music, Atago Provides Playlist Support and 24/7 Looping Queue ready to use.

III. Getting Started

Invite:

Invite Atago to your Server now! Found a bug or Need Help with Atago? Support Server


Setup:

Setup Atago! Set Welcome Channel, Welcome Message, Welcome Description, Boost Channel, Boost Message, Boost Role, and More!

at!welcomechannel [@channel]
at!welcomemessage [Message] ({{user}}, {{count}})
at!welcomedescription [Message]({{member}}, {{count}})
at!boostchannel [@channel]
at!boostrole [@role] (For Testing use a testing role and add to a user to see if its working)
at!boostmessage [Message]({{user}})
at!modlog [@channel]
at!announcelevel [off|on]
at!announcetype [channel|dm]
at!welcomemembers [on|off]
at!prefix [prefix]
